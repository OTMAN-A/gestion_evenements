# Gestion Événements

Ce projet est une application de gestion d'événements développée avec PHP et XAMPP.

## Fonctionnalités

- Création, modification et suppression d'événements
- Gestion des participants
- Interface utilisateur simple et intuitive

## Prérequis

- [XAMPP](https://www.apachefriends.org/fr/index.html)
- PHP >= 7.4
- MySQL

## Technologies utilisées

PHP 8+

MySQL 5.7 / 8.0

HTML5 / CSS3

Bootstrap 5

JavaScript (pour les interactions simples)

Architecture MVC

## Installation

1. Clonez ce dépôt dans le dossier `htdocs` de XAMPP :
    ```bash
    git clone <url-du-repo> gestion_evenements
    ```
2. Démarrez Apache et MySQL via le panneau de contrôle XAMPP.
3. Importez la base de données depuis le fichier `database.sql` (s'il existe).
4. Accédez à l'application via `http://localhost/gestion_evenements`.


## Utilisation

- Ajoutez, modifiez ou supprimez des événements via l'interface web.
- Gérez les participants pour chaque événement.
